# TradeIQ | Smart Investing Platform

TradeIQ is an advanced stock market intelligence and portfolio management platform. It leverages a multi-agent AI architecture to synthesize real-time market data, news sentiment, and macroeconomic indicators into actionable insights.

---

## 🏗️ System Architecture

TradeIQ operates on a high-concurrency **Broad Parallel Flow** that merges traditional machine learning with modern LLM reasoning.

## 🛠️ Tech Stack

* **Frontend:** ReactJS, Tailwind CSS
* **Backend:** FastAPI (Python)
* **Mobile:** Jetpack Compose (Kotlin)
* **AI/ML:** BERT (Sentiment), Mamba (Forecasting), LLM Orchestration
* **Automation:** n8n workflow orchestration
